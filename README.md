# AI 자격증 모의고사 & 학습 대시보드

**5개 AI 관련 자격시험**을 하나로 대비하는 웹 대시보드입니다. 상단 칩으로 시험을 전환하며, 시험별로 점수·오답·기록이 따로 관리됩니다.
외부 라이브러리·서버 없이 동작하는 **단일 정적 HTML** 파일 하나로 구성됩니다. (상태는 브라우저 `localStorage`에 저장)

## 🎓 수록 시험 (5종)
| 시험 | 발급 | 실전 모의 | 문제은행 |
|---|---|---|---|
| **AI-POT 2급** (AI 프롬프트 활용능력) | KPC | 40문항·60분·70점 | 객관식 230 + 실습형 60 |
| **AIBT** (AI 비즈니스 활용능력) | KPC | 40문항·60분·70점 | 48 |
| **Azure AI-900** (AI Fundamentals) | Microsoft | 40문항·45분·70% | 50 |
| **ADsP** (데이터분석 준전문가·국가공인) | K-DATA | 50문항·90분·60점 | 60 |
| **AICE** (Associate·국가공인) | KT·한경 | 40문항·60분·70% | 48 |

## ✨ 주요 기능
- **사용자 프로필** — 정덕진 / 정상옥 선택(로그인 불필요), 시험별·사용자별 데이터 분리
- **실전/미니 모의고사** — 각 시험 규격에 맞춘 문항수·시간·합격선, CBT식 네비게이터·자동 채점
- **영역별 학습** / **오답노트** / **실습형 연습**(AI-POT) / **시험 정보**(일정·구성)
- **다양성 로테이션** — 매 회차 *아직 안 푼 문항을 우선* 출제, 학습 진척률 추적

## 📚 문제은행
- 5개 시험 합계 **객관식 436문항 + 실습형 60문항**
- ※ 공개된 출제기준·유형을 바탕으로 제작한 **학습용 예상문제**이며, 실제 기출문제가 아닙니다.

## ▶ 사용법
- **로컬:** `index.html`을 브라우저로 열기 (더블클릭)
- **URL:** 아래 방법으로 배포하면 주소만으로 어디서나 실행

## 🚀 배포 (URL로 실행하기)
정적 파일이라 아무 정적 호스팅에나 올리면 됩니다.

### GitHub Pages (무료·추천)
1. 이 저장소를 GitHub에 push (public)
2. 저장소 **Settings → Pages → Source: `main` 브랜치 / `/ (root)`** 선택 후 저장
3. 1~2분 뒤 `https://<사용자명>.github.io/<저장소명>/` 에서 접속

### 대안
- **Netlify / Cloudflare Pages / Vercel:** 저장소 연결 또는 폴더 드래그&드롭 → 즉시 URL 발급
