# AI-POT 2급 모의고사 & 학습 대시보드

한국생산성본부(KPC) **AI 프롬프트 활용능력(AI-POT) 2급** 시험 대비용 웹 대시보드입니다.
외부 라이브러리·서버 없이 동작하는 **단일 정적 HTML** 파일 하나로 구성되어 있습니다. (상태는 브라우저 `localStorage`에 저장)

## ✨ 주요 기능
- **실전 모의고사** — 40문항 / 60분 카운트다운 / 100점 만점 70점 합격, CBT식 문제 네비게이터·자동 채점
- **미니 모의고사** — 20문항 / 30분
- **영역별 학습** — 5개 평가영역별 문제 + 즉시 해설
- **실습형 연습** — 프롬프트 작성·개선·단답·오류찾기 (모범답안 + 채점 포인트 자가채점)
- **오답노트** — 틀린 문제 모아보기
- **다양성 로테이션** — 매 회차 *아직 안 푼 문항을 우선* 출제해 시작할 때마다 새로운 문제, 학습 진척률 추적

## 📚 문제은행
- 객관식 **230문항** (5개 평가영역 × 46)
- 실습형 **60문항** (작성형·개선형·단답·오류찾기)
- ※ 공개된 출제기준·유형을 바탕으로 제작한 **학습용 예상문제**이며, 실제 기출문제가 아닙니다.

## ▶ 사용법
- **로컬:** `index.html`을 브라우저로 열기 (더블클릭)
- **URL:** 아래 방법으로 배포하면 주소만으로 어디서나 실행

## 🚀 배포 (URL로 실행하기)
정적 파일이라 아무 정적 호스팅에나 올리면 됩니다.

### GitHub Pages (무료·추천)
1. 이 저장소를 GitHub에 push (public)
2. 저장소 **Settings → Pages → Source: `main` 브랜치 / `/ (root)`** 선택 후 저장
3. 1~2분 뒤 `https://<사용자명>.github.io/<저장소명>/` 에서 접속

### 대안
- **Netlify / Cloudflare Pages / Vercel:** 저장소 연결 또는 폴더 드래그&드롭 → 즉시 URL 발급
